## M6.1 簡單處理器 — Leros（單週期 / FSMD）

| datapath 元件 | 用什麼做 | 章節 |
|---------------|----------|------|
| PC | `RegInit` + 加法 | M1.5 |
| Instruction Memory | `SyncReadMem` / ROM | M2.5、Part 7 |
| Register File | `Reg(Vec)` | M1.6 |
| ALU | `switch(op)` | M2.1 |
| Control | FSM / FSMD | M3 |

```scala
class Leros extends Module {
  val io  = IO(new Bundle { val dout = Output(UInt(32.W)) })
  val pc  = RegInit(0.U(16.W)); val acc = RegInit(0.U(32.W))
  val prog = VecInit(Array(0.U(32.W) /* 機器碼，或讀檔生成 */))
  val instr  = prog(pc)
  val opcode = instr(31, 24); val imm = instr(15, 0)
  switch(opcode) {
    is(0x08.U){ acc := imm }         // load imm
    is(0x09.U){ acc := acc + imm }   // addi
    is(0x0a.U){ acc := acc - imm }   // subi
  }
  pc := pc + 1.U
  io.dout := acc
}
```
⚠️ **陷阱**：`SyncReadMem` 讀有一拍延遲；單週期教學版可用組合讀 `Mem` 簡化時序。

對應：§14。

## M6.2 RISC-V 3-stage Pipeline — Wildcat

```
[1] Fetch            取指、PC+4
[2] Decode+RegRead   解碼、讀 register file
[3] Execute+Mem      ALU、記憶體、寫回
```

```scala
class DecodeExec extends Bundle {     // pipeline register payload（不帶方向）
  val rd = UInt(5.W); val rs1Val, rs2Val, imm = UInt(32.W)
  val aluOp = UInt(4.W); val regWrite = Bool()
}

class Wildcat extends Module {
  val io = IO(new Bundle { val dbg = Output(UInt(32.W)) })
  // Stage 1: Fetch
  val pc = RegInit(0.U(32.W)); val imem = SyncReadMem(1024, UInt(32.W))
  val instr = imem.read(pc >> 2); pc := pc + 4.U
  // Stage 2: Decode + RegRead
  val rf = Reg(Vec(32, UInt(32.W)))
  val de = Wire(new DecodeExec)
  de.rd := instr(11,7); de.rs1Val := rf(instr(19,15)); de.rs2Val := rf(instr(24,20))
  de.aluOp := instr(14,12); de.imm := instr(31,20); de.regWrite := true.B
  val deReg = RegNext(de)            // ★ pipeline register
  // Stage 3: Execute + Writeback
  val aluY = WireDefault(0.U(32.W))
  switch(deReg.aluOp){ is(0.U){ aluY := deReg.rs1Val + deReg.rs2Val }; is(7.U){ aluY := deReg.rs1Val & deReg.rs2Val } }
  when(deReg.regWrite && deReg.rd =/= 0.U){ rf(deReg.rd) := aluY }   // x0 恆 0
  io.dbg := aluY
}
```

| 概念 | 處理 |
|------|------|
| stage 隔離 | stage 間放 `RegNext(bundle)` / `Valid[T]` |
| `x0` 恆 0 | 寫回時 `rd =/= 0.U` 才寫 |
| imm sign-extend | `Cat(Fill(20, inst(31)), inst(31,20))` |
| **data hazard** | 後級結果未寫回 → forwarding / stall |
| **control hazard** | branch 改 PC → flush（插 bubble） |

⚠️ **陷阱（pipeline 必踩）**：data hazard（相依指令讀到舊值）、branch hazard（分支成立要 flush）。
教學版可先「假設無相依」，進階版加 forwarding/stall。

對應：§15。

## M6.3 微架構資料結構（朝 in-order+ / OoO 鋪路）

### `MicroOp` — pipeline 核心結構
```scala
class MicroOp extends Bundle {
  val valid = Bool()
  val pc, inst, imm = UInt(32.W)
  val ctrl = new DecodeSignals
  val rs1, rs2, rd  = UInt(5.W)
}
```
> 保留 `inst` 對 debug / trace / difftest 很有用。bubble 用 `MicroOp.bubble`（見 M3.5），不要塞 NOP。

### Decoder 輸出包成 Bundle（別散落 wire）
```scala
class DecodeSignals extends Bundle {
  val regWrite, memRead, memWrite = Bool()
  val aluOp  = UInt(4.W)
  val op1Sel, op2Sel = UInt(2.W)
}
```

### Exception / interrupt 用 `Valid[Payload]`
```scala
class ExceptionInfo extends Bundle { val cause = UInt(5.W); val tval, pc = UInt(32.W) }
val exception = Valid(new ExceptionInfo)     // ✗ 不要只用一個 Bool
```

### Memory response 用 tag（支援亂序回應）
```scala
class MemReq  extends Bundle { val addr = UInt(32.W); val tag = UInt(tagBits.W) }
class MemResp extends Bundle { val data = UInt(32.W); val tag = UInt(tagBits.W) }
```
用於：OoO memory response、MSHR、multicycle EU、TLB、BP query。

對應：§14–§15 延伸。

## M6.4 Flush 管理：epoch / branch mask

### Epoch（sequence number）殺 stale packet
```scala
val epoch = RegInit(false.B)
when(branchMispredict){ epoch := !epoch }
class FetchPacket extends Bundle { val pc, inst = UInt(32.W); val epoch = Bool() }
val killed = packet.epoch =/= currentEpoch    // 比全域廣播 flush 乾淨
```

### Branch mask（speculation 追蹤）
```scala
val branchMask = UInt(nBranches.W)
uop.branchMask := uop.branchMask & ~resolvedBranchOH         // resolve 清掉該 branch
val killed     = (uop.branchMask & mispredBranchOH).orR      // mispredict kill
```

## M6.6 Performance counters

val cycle   = RegInit(0.U(64.W)); cycle := cycle + 1.U
val instret = RegInit(0.U(64.W)); when(io.commit.fire){ instret := instret + 1.U }

class CorePerf extends Bundle {
  val cycle, instret = UInt(64.W)
  val frontendStall, backendStall, loadUseStall, branchFlush, cacheMiss = UInt(64.W)

## M6.7（選修）整合 legacy Verilog（BlackBox 概覽，詳見 Part 7）
```scala
import chisel3.experimental.{IntParam, ExtModule}
class MyVerilogIP extends ExtModule(Map("WIDTH" -> IntParam(8))) {
  val clk = IO(Input(Clock())); val a = IO(Input(UInt(8.W))); val y = IO(Output(UInt(8.W)))
}
```
⚠️ **陷阱**：port 名稱/型別必須與外部 Verilog 完全一致；clock 顯式 `Input(Clock())`。

**練習**
- ★ 為 Leros 加 branch 指令，跑一段迴圈程式。
- ★★ 為 Wildcat 加 load/store 接 data memory。
- ★★★ 為 Wildcat 實作 forwarding 解 data hazard，並用 commit trace + ChiselSim 驗證一段有相依的程式（Part 8）。
