## A. Chisel ↔ Verilog / VHDL 對照

| 概念 | Chisel | Verilog |
|------|--------|---------|
| 模組 | `class M extends Module` | `module M ... endmodule` |
| 輸入/輸出 | `Input(UInt(8.W))` / `Output(...)` | `input/output [7:0]` |
| wire | `Wire(UInt(8.W))` / `WireDefault(0.U)` | `wire [7:0]` |
| 暫存器 | `RegInit(0.U(8.W))` | `reg [7:0]` + `always @(posedge clk)` |
| 連線 | `y := a & b` | `assign y = a & b` |
| 多路 | `when/.elsewhen/.otherwise` | `if/else` in `always` |
| case | `switch(x){ is(...){} }` | `case (x) ... endcase` |
| 實例化 | `Module(new Sub)` | `Sub u_sub(...)` |
| clock/reset | **隱含自動**（`RawModule` 可手控） | 必須手寫 |
| 參數化 | Scala（case class / 型別 / Diplomacy） | `parameter` / generate |
| **生成器** | 整個 Scala（`for`/`map`/遞迴/ADT） | 有限（generate） |

## C. 陷阱 + 慣用法總表（★列印隨身）

### C-1 基礎陷阱
| # | 陷阱 | 錯 | 對 |
|---|------|----|----|
| 1 | 寬度漏 `.W` | `1.U(32)`（取第 32 位=0） | `1.U(32.W)` |
| 2 | 比較運算子 | Scala `==` | Chisel `===` / `=/=` |
| 3 | 生成 vs 硬體 | 用 `if`/`for` 想做 runtime | runtime 用 `when`/`Mux` |
| 4 | latch | `when` 沒 default | `WireDefault` 或 `.otherwise` |
| 5 | last-connect | 同線多次 `:=` | 後者覆蓋（刻意才用） |
| 6 | Reg 無 reset | `Reg(...)` 期望有初值 | `RegInit(...)` |
| 7 | Vec 誤用 | `Wire(Vec)` 當暫存器 | `Reg(Vec(...))` |
| 8 | 非同步輸入 | 直接進邏輯 | 先 2-FF synchronizer |

### C-2 乾淨 RTL / 生成器慣用法
| # | 情境 | 慣用法 |
|---|------|--------|
| 9 | payload bundle | **不帶方向**；方向交給 `Decoupled`/`Flipped`/IO |
| 10 | 避免未連接 | `WireDefault(default)` + 局部 override |
| 11 | invalid payload | 可 `DontCare`；**valid payload / external IO 不可** |
| 12 | bubble | `uop.valid := false.B`，**不要**塞 NOP |
| 13 | pipeline reg | 包成 `Valid[T]` / `RegEnable(next, init, en)` |
| 14 | control 優先序 | 寫死 `kill > replay > stall > advance` |
| 15 | default 值 | companion object（`MicroOp.bubble`），非 `0.U.asTypeOf`（zero 未必合法） |
| 16 | large array | 只 reset valid bits，payload 免 reset |
| 17 | valid vs payload | 兩個 array 分開（ROB/IQ/MSHR/LSQ） |
| 18 | adder/min tree | `reduceTree`（O(log n)），非 `reduce`（O(n)） |
| 19 | 選擇器 | small=`MuxLookup`、one-hot=`Mux1H`、優先=`MuxCase`、FSM=`switch`、ISA=`BitPat` |
| 20 | 參數 | `case class Params` + `require` + derived param 集中 |
| 21 | 可選硬體/選擇 | `Option`（optional）/ sealed trait ADT（互斥選擇） |
| 22 | 泛型模組 | `[T <: Data](gen: T)` + `chiselTypeOf` |

### C-3 流控 / 微架構 / SoC
| # | 陷阱 | 對策 |
|---|------|------|
| 23 | `Decoupled` ≠ 有 register | 要 buffer 用 `Queue`；`Decoupled` 只是 protocol |
| 24 | fire vs valid | 狀態更新掛 `.fire`（valid && ready） |
| 25 | combinational loop | `valid` 不可組合相依於 `ready` |
| 26 | ready critical path | 插 `SkidBuffer` / `TLBuffer` |
| 27 | exception/event | 用 `Valid[Payload]`，非單一 `Bool` |
| 28 | CDC | Chisel **不自動**；用 `AsyncQueue`/synchronizer |
| 29 | BlackBox port | 名稱/型別與外部 Verilog **完全一致** |
| 30 | SyncReadMem | 讀延遲一拍；metadata 用 `RegEnable` 對齊；RUW 自理 |
| 31 | dontTouch 過量 | 優先正式 debug IO / Probe / Layer |
| 32 | sim-only logic | 放 TestHarness，不要塞進 core RTL |
| 33 | **測試框架** | 新專案用 **ChiselSim**，非 `chiseltest` |

## D. 版本注記（Chisel 7 / Scala 3 / ChiselSim）

| 主題 | 重點 |
|------|------|
| Chisel 7.0（2025-09） | 支援 Scala 3（亦相容 2.13）；本 Handout 用 7.13.0 |
| 後端 | MFC（CIRCT / firtool），解鎖 Layer / Probe / LTL |
| 測試 | **ChiselSim** 取代 **ChiselTest**（deprecated，官方不維護） |
| Verilog | `circt.stage.ChiselStage.emitSystemVerilog(...)` |
| ChiselEnum / MuxLookup | 已穩定；`MuxLookup(k, d)(mapping)`（curried） |

### ChiselTest → ChiselSim 遷移速查
| 舊 | 新 |
|----|----|
| `import chiseltest._` | `import chisel3.simulator.EphemeralSimulator._` 或 `with ...scalatest.ChiselSim` |
| `test(new X){ c => }` | `simulate(new X){ dut => }` |
| `WriteVcdAnnotation` | 旗標 `-DemitVcd=1` |
| poke/peek/expect/step | **相同**（`peek().litValue` 取數值） |

## E. 核心 Chisel vs Rocket/Chipyard API（避免混淆）

| 屬於 core Chisel | 屬於 Rocket Chip / Chipyard |
|------------------|------------------------------|
| `Module`/`Bundle`/`Vec`/`Reg`/`Wire` | `LazyModule`/`LazyModuleImp` |
| `Decoupled`/`Queue`/`Arbiter` | `Parameters`/`Field`/`Config` fragment |
| `SyncReadMem`/`Mem`/`BlackBox` | Diplomacy / `TLNode`/`TLXbar`/`TLBuffer` |
| `assert`/`cover`/`printf` | `regmap`/`RegField`/`TLRegisterRouter` |
| `Layer`/`Probe`/`DataView`（Chisel 5+） | IOBinders/HarnessBinders、RoCC、`site/here/up` |
